# **App Name**: Commitment Journey

## Core Features:

- Commitment Button: A prominent button on the main screen that initiates the commitment process.
- Double Confirmation: A double confirmation screen to ensure the user is fully aware and committed.
- Details Input Screen: A letter-style UI for entering Instagram ID and Name.
- Confirmation Message: A message confirming that the user will be contacted later.

## Style Guidelines:

- Primary color: Soft white (#F0F0F0) to create a sense of clarity.
- Background color: Ash black (#2E2E2E), slightly desaturated from true black, providing contrast.
- Accent color: Light gray (#A9A9A9) for interactive elements to maintain a neutral aesthetic.
- Body and headline font: 'Inter', a sans-serif, for clear, readable text.
- A minimalist layout with a focus on the central commitment button.
- Subtle animations on button clicks to enhance user experience.
